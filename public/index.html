How it Works

postAR was built using React Native and runs MongoDB on the backend. User posts are sent to the database with their coordinates which are stored along with the contents of the post. When a user opens the camera view, the app requests all posts within a certain radius and, using some cool math, displays them on the phone's screen in approximately the correct position.
