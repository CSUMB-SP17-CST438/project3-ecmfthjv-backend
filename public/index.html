How it Works

postAR was built using React Native and runs MongoDB on the backend. User posts are sent to the database with their coordinates which are stored along with the contents of the post. When a user opens the map view, the app requests all posts within a certain radius and displays them in the correct position on the 2D map. The next steps including turning this experince into one that can only be experienced through Augmented Reality.
